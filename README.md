Tecnologías utilizadas
Front-End
- *HTML5* – Estructura semántica
- *CSS3* – Layout, animaciones y responsive design
- *JavaScript (Vanilla JS)* – Interactividad, i18n y lógica básica

Funcionalidades destacadas
-  Sistema de internacionalización *(i18n)* Español / Inglés
-  Diseño *responsive* (desktop y mobile)
-  Formulario de contacto funcional
-  Persistencia de idioma usando localStorage

Herramientas y servicios

- *Firebase Hosting* – Despliegue del sitio
- *Formspree* – Envío de formularios sin backend
- *Ngrok* – Pruebas de responsive y testing local
- *Git & GitHub* – Control de versiones
- *VS Code* – Editor principal
- *Sublime Text* – Uso previo
